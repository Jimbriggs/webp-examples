1 2005 June 09

If you are marking "study" projects using the on line system, you should be aware that there is a problem with the fourth category: "Primary research & results (if any)".
If the student did not do any primary research (and this is NOT a requirement in the UD) and thus you leave this blank, then the system will give you an error message and you cannot proceed until you assign a mark -so you must then go 'Back' and assign a nominal mark in this category and recalculate.
However, strangely if you then take out the mark for this category entirely and recalculate again the system does not give you an error message but instead simply ignores the fact that you have taken out this mark and automatically reassigns the mark you gave previously! (without notification to the user it has done this). Out of interest this holds true for ALL categories on both the PJE30 and PJS30 forms.

It needs the ActionForm's reset method to clear all the fields corresponding to checkboxes.
